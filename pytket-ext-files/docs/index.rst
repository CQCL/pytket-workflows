pytket-TODOEXTNAME
=============

``pytket-TODOEXTNAME`` is an extension to ``pytket`` that allows ``pytket`` circuits to
be executed on TODOPYTKETEXT.

``pytket-TODOEXTNAME`` is compatible with Python versions 3.10 to 3.13 on Linux,
MacOS and Windows. To install, run:

::

    pip install pytket-TODOEXTNAME

.. note::   Requires an TODOPYTKETEXT account.
